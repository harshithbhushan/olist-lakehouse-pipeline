# Olist E-Commerce Lakehouse
**Status:** Active Development | **Role:** Data Engineer

## Project Objective
A production-grade ELT pipeline processing Brazilian E-Commerce data using the Medallion Architecture (Bronze -> Silver -> Gold).
Designed to handle data quality checks, schema evolution, and automated orchestration.

## Tech Stack
* **Cloud:** Snowflake (Standard Edition - AWS)
* **Orchestration:** Apache Airflow (Dockerized)
* **Transformation:** dbt Core
* **Language:** Python 3.12, SQL
